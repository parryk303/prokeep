# Set Up Commands
> Run the following commands from the root of this repo
```
npm i
```
```
npm start
```
```
<navigate to localhost:3000 in your browser>
```

# Testing Commands
> Run the following commands from the root of this repo
```
npm run test
```
```
enter option <a> to run all tests
```
